"SCOOP TAB" is a online ice cream delivery and parlor management system. There is 4 user of the system (owner,manager,employee,customer). 
By using Scoop Tab admin can store information about the stock of products, customer orders, daily business, billing and staff of a ice cream parlor.
Customer can order by registering their details.


List of softwares that was used to create the system : 
1/ Visual Studio 2019 
2/ SQL Server 2014 Management Studio
